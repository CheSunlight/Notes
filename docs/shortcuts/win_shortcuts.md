# Windows 快捷键

## 默认

- **win + D**: 显示桌面
- **win + E**: 打开资源管理器
- **win + I**：打开系统设置
- **win + S**: 搜索
- **win + 方向键**：向对应方向分屏

## 自定义

- **Ctrl + Alt + S**：打开 VSCode
- **Ctrl + Alt + E**：打开 Edge
- **Ctrl + Alt + C**：打开 Clash
- **Ctrl + Alt + F**：打开飞书
- **Ctrl + Alt + Z**：打开 Zotero
- **Ctrl + Alt + W**：打开微信
