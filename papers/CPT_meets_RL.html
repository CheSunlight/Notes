Contributions

  • The first to define and investigate human-centered RL problem.
  • The first to combine CPT with RL.

Human-centered RL Problem

  • Agent controls a system to produce returns that are maximally aligned with the preferences of one or possibly multiple humans.
  • Preferences of rational agents facing decisions with stochastic outcomes can be modeled using expected utilities.
  • Here the authors use CPT to model the stochastic outcomes.

Function Approximation and Boltzmann Policy

Function Approximation

  • Note that in QQQ-learning, the policy is dependent on the QQQ-table, which contains all Q(s, a)Q(s,\ a)Q(s, a) for each state sss and each action aaa. At some state sss, the agent choose the action with biggest QQQ value. The dimension of the QQQ-table is S×AS \times AS×A where SSS is the num of states and AAA is the num of feasible actions.
  • In problems with high-dimension of states and actions, QQQ-table becomes computationally expensive. Thus, we use function approximation to approximate the QQQ funtion:
    Q(s, a)≈θTϕs, aQ(s,\ a) \approx \theta^{\mathsf{T}} \phi_{s,\ a}Q(s, a)≈θTϕs, a​ where ϕs, a\phi_{s,\ a}ϕs, a​ is a ddd-dimensional vector describing the state-action feature, e.g., ddd can be S+AS+AS+A if we construct features on each states and actions separately instead of considering their combinations.

Boltzmann Policy

  • Instead of choosing action with the biggest QQQ value, Boltzmann policy makes the QQQ values to be probs under some state.
